## 概要
10000から3秒ごとに繰り返した回数と同じ数を引いていきます。  

## 操作方法
    $ make  
    $ sudo insmod myled.ko  
    $ sudo chmod 666 /dev/myled0  
    $ echo [0∼2] > /dev/myled0
